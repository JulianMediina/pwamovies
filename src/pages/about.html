🎬 Acerca de CineCache

¿Qué es CineCache?

CineCache es una Progressive Web App (PWA) diseñada para demostrar las mejores prácticas en caching y manejo offline. Busca películas usando la API OMDb y las cachea inteligentemente para que funcione sin conexión a internet.

5 Estrategias de Caching

🔵 Cache First (App Shell)

HTML, CSS, JS cacheados durante install

🟢 Network First (API)

Datos frescos con fallback a caché offline

🟡 Stale While Revalidate (Imágenes)

Caché rápido + actualización en bg

Característica: "Leer Después"

Guarda tus películas favoritas en localStorage para leerlas después. Los datos persisten incluso después de cerrar la app.
